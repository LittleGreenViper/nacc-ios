DESCRIPTION

This is an extremely basic toy app that is for use by NA members for calculating their cleantime -not their Recovery. It asks the user to enter their clean date in the app, then displays a textual breakdown of their cleantime to date, along with a string of keytags.

REQUIREMENTS

This is an iOS app, made for iPhones and iPads. It is only available via the Apple App Store, and requires iOS 11.0 or above.

The Watch component requires watchOS 5.0 or above.

INSTALLATION

This will be installed via the Apple App Store, and will be a free app.
